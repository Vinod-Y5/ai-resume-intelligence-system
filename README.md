# AI Resume Intelligence System

An end-to-end AI-powered system that scrapes job descriptions, extracts skills using NLP, generates tailored resume content using LLMs, and auto-fills LaTeX resume templates.

## Tech Stack
- Python
- NLP (spaCy)
- LLMs (OpenAI)
- Web Scraping
- LaTeX Automation

## Features
- Job description analysis
- Skill extraction
- AI-based resume generation
- ATS-friendly LaTeX resumes

## Usage
```bash
python main.py
